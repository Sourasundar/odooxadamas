# 🌌 OdooXadamas

Welcome to **OdooXadamas**, a modern web application currently undergoing a major UI/UX transformation based on the **Antigravity Design** principles.

## 🚀 Ongoing Work: Antigravity UI Refactoring
We are actively refactoring the frontend to implement a state-of-the-art **Antigravity Design System**. Our focus includes:

- **Glassmorphism & Spatial Depth:** Moving away from traditional flat UI to a rich, multi-layered glassmorphic design using translucent panels and deep shadows.
- **Advanced Motion & Animation:** Integrating `GSAP` (GreenSock) and `@gsap/react` for smooth, staggered entrances and scroll-triggered animations.
- **3D Interactions:** Implementing isometric interactive cards (`rotateX/rotateY` transforms), parallax scroll effects, and weightless drop-shadows.
- **Micro-interactions & Polish:** Experimenting with magnetic buttons, custom cursors, dynamic dark/light spatial themes, and premium page transitions.

## 🛠 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS v4, GSAP
- **Backend:** Backend API Service

## 📂 Project Structure
- `/frontend` - The React application featuring the new Antigravity UI.
- `/backend` - The server-side API application.

## 🏃 Getting Started

To run the project locally, start both the frontend and backend development servers concurrently.

```bash
# Start the Backend Server
cd backend
npm install
npm run dev
```

```bash
# Start the Frontend Application
cd frontend
npm install
npm run dev
```
